About GitHub:
Millions of developers and companies build, ship, and maintain their software on GitHub—the largest and most advanced development platform in the world.
Краткое замечание: git init и git clone легко спутать, поскольку обе команды можно использовать для инициализации нового репозитория Git. Однако команда git clone зависит от git init и предназначена для создания копии существующего репозитория. Выполнение git clone сначала вызывает git init, чтобы создать новый репозиторий, затем копирует данные из существующего репозитория и извлекает новый набор рабочих файлов.

- присутсвие .gitignore
- удаление уже существующего файла и переименование
